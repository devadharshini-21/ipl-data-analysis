# 🏏 IPL Data Analysis Dashboard

An interactive IPL analytics project built using Python and Streamlit to explore match trends, team dominance, player performances, toss impact, and city-wise statistics from historical IPL datasets.

---

## 📊 Highlights

* Visualized IPL match insights using charts and graphs
* Explored team and player performance trends
* Analyzed toss decisions and match outcomes
* Built an interactive dashboard with Streamlit
* Worked with real-world IPL datasets

---

## 🛠️ Built With

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

## 🔍 Insights

* Mumbai Indians and Chennai Super Kings consistently dominate IPL wins.
* Certain players repeatedly secure Player of the Match awards, highlighting consistent performance.
* Toss decisions show noticeable influence on match outcomes in some venues.
* Metro cities host a major share of IPL matches across seasons.
